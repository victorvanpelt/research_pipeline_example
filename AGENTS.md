# AGENTS.md — context for AI coding agents

This repository is a template for a reproducible research pipeline. Data flows through numbered
folders; everything downstream of the raw data is regenerated by code.

## Rules

- Never modify anything in `0_data/`. Raw data is read-only.
- Never edit files in `2_process/` or `3_output/` by hand: they are build artifacts, regenerated
  by the pipeline.
- Make changes in `1_code/` (analysis) or `4_drafts/` (paper and slide sources) only.
- The three analysis engines must stay in sync: `1_code/code.do` (Stata), `1_code/code.r` (R), and
  `1_code/code.qmd` (Quarto) produce identical results.
- Do not commit credentials; use environment variables.

## Build

- Full rebuild: `make`
- Single steps: `make r`, `make stata`, `make quarto`, `make paper`, `make slides`
- Reset intermediates and outputs: `make clean`

## Verification

A change is done only when `make clean` followed by `make` completes without errors and every
number in the `4_drafts/` outputs comes from the rerun pipeline, not from hand edits.
