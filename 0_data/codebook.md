# Codebook: gen_ai_earnings.csv

Synthetic example data, hand-authored for teaching. All values are invented; no real firms or people. Reuse freely.

**Unit of observation:** one synthetic firm-observation per row (N = 200). Plain CSV with a header row.

| Variable | Type | Definition |
|----------|------|------------|
| `id` | integer | Observation identifier (1–200) |
| `gen_ai` | indicator | 1 = uses generative AI, 0 = does not |
| `earnings` | numeric | Synthetic earnings measure, in arbitrary units (negative values occur) |

If you replace the example with your own data, document it here the same way: define every variable, and record the source, the citation, the download date, and how someone else can obtain the file.
